# GOFIT - Asistente de Entrenamiento y Nutrición

GOFIT es una microapp web diseñada para acompañar a los usuarios en su día a día deportivo, actuando como un entrenador personal y nutricionista de bolsillo.

## 🚀 Tecnologías
*   React
*   TypeScript
*   Vite
*   Tailwind CSS
*   React Router DOM

## 📋 Gestión del Proyecto
El desarrollo de este proyecto se gestiona utilizando metodologías ágiles.
Puedes consultar:
*   [Tablero Kanban de Trello (Documentado localmente)](docs/trello.md)
*   [Metodología y Gestión del Proyecto](docs/project-management.md)

## 💻 Instalación y Uso Local

```bash
# Clonar el repositorio
git clone https://github.com/serradogee/GOFIT.git

# Entrar al directorio
cd GOFIT

# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```
